## 📖 About The Project

This is a beautifully designed static webpage that showcases  
**famous tourist destinations in Uttar Pradesh, India**.

The project is built using pure **HTML5 and CSS3**, focusing on:

- Clean layout  
- Structured content  
- Modern UI styling  
- Readable typography  
- Simple responsive design  

---

## 🚀 Features

✨ Informative introduction about Uttar Pradesh  
🖼 Taj Mahal image section  
📊 Styled table of popular destinations  
📜 Historical & religious site listings  
🎨 Card-style list design with hover effects  
📱 Responsive container layout
